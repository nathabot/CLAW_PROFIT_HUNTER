## How to Set Up Safe Automation

### Step 1: Mark Account as Automated

**Option A: Twitter Settings**
1. Go to Settings → Your account
2. Find " automation" or "bot" settings
3. Mark account as automated

**Option B: Add Disclosure in Bio**
```
Bio: AI trading agent | Automated by @usearound
```

### Step 2: Safe Automation Rules

| Action | Safe Limit | Danger |
|--------|------------|--------|
| Posts/day | 10-20 | 50+ |
| Likes/day | 50-100 | 200+ |
| Retweets/day | 20-50 | 100+ |
| Follows/day | 30-50 | 100+ |
| Unfollows/day | 30-50 | 100+ |
| Delay between actions | 30-120s | <10s |

### Step 3: Natural Behavior Patterns

**Do:**
- Vary posting times
- Mix content types (text, images, threads)
- Respond to some mentions
- Have natural gaps in activity

**Don't:**
- Post at exact intervals
- Like all tweets from one user
- Follow/unfollow same person
- Mass engage with trending
