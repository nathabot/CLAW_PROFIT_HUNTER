# Trading Strategy Intelligence Database

A comprehensive SQLite database for storing, analyzing, and exporting trading strategies, signals, smart money data, and market context.

## 📁 Files

| File | Description |
|------|-------------|
| `schema.sql` | Database schema definition with indexes and constraints |
| `db.js` | Node.js helper module with CRUD operations |
| `init-db.js` | Database initialization script |
| `test-db.js` | Test suite and usage examples |
| `package.json` | NPM configuration |

## 🗄️ Database Location

```
/root/trading-bot/strategy-intelligence.db
```

## 📊 Schema Overview

### Tables

1. **strategies** - Trading strategies with rules and parameters
2. **signals** - Trading signals generated by strategies
3. **smart_money** - Whale and smart money transaction tracking
4. **market_context** - Market sentiment and conditions snapshot
5. **strategy_performance** - Aggregated performance metrics

## 💻 Usage Examples

### Connect and Query

```javascript
const { getDatabase } = require('./db.js');

const db = getDatabase();
await db.connect();

// Get dashboard data
const dashboard = await db.getDashboardData();

// Get active strategies
const strategies = await db.getStrategies({ minConfidence: 70 });

// Get recent signals
const signals = await db.getSignals({ limit: 10 });

await db.close();
```

### Create a Strategy

```javascript
const strategyId = await db.createStrategy({
    name: 'Twitter Momentum Alpha',
    source: 'twitter',
    type: 'momentum',
    description: 'Captures momentum from viral Twitter mentions',
    entry_rules: {
        min_mentions_per_hour: 50,
        sentiment_threshold: 0.7,
        volume_spike: 2.0
    },
    exit_rules: {
        take_profit: 0.15,
        stop_loss: 0.07
    },
    risk_params: {
        position_size: 0.05,
        max_positions: 5
    },
    confidence_score: 82.5
});
```

### Create a Signal

```javascript
const signalId = await db.createSignal({
    token_symbol: 'BTC',
    strategy_id: 1,
    signal_type: 'buy',
    entry_price: 98500.00,
    target_price: 108000.00,
    stop_loss: 94500.00,
    source: 'onchain_scanner',
    confidence: 85.5,
    notes: 'Whale accumulation detected'
});
```

### Record Smart Money

```javascript
await db.recordSmartMoney({
    wallet_address: '0x742d35...',
    entity_name: 'Jump Trading',
    token_symbol: 'BTC',
    action: 'buy',
    amount: 150.5,
    amount_usd: 14800000,
    tx_hash: '0xabc123...',
    chain: 'ethereum',
    block_number: 18234567
});
```

### Export for Trading Division

```javascript
const exportData = await db.exportForTradingDivision({
    signalLimit: 50,
    strategyLimit: 20,
    includeSmartMoney: true,
    includeMarketContext: true
});

// Returns structured JSON for trading bot consumption
```

## 📈 Available Methods

### Strategies
- `createStrategy(strategy)` - Create new strategy
- `getStrategy(id)` - Get strategy by ID
- `getStrategies(filters)` - Get strategies with filtering
- `updateStrategy(id, updates)` - Update strategy
- `deleteStrategy(id)` - Delete strategy

### Signals
- `createSignal(signal)` - Create new signal
- `getSignal(id)` - Get signal by ID
- `getSignals(filters)` - Get signals with filtering
- `markSignalExecuted(id, execution)` - Mark signal as executed
- `getPendingSignals()` - Get all pending signals

### Smart Money
- `recordSmartMoney(transaction)` - Record transaction
- `getSmartMoney(filters)` - Get smart money activity
- `getSmartMoneySummary()` - Get summary by token
- `getWhaleMovements(minUsd, hours)` - Get whale movements

### Market Context
- `recordMarketContext(context)` - Record market snapshot
- `getLatestMarketContext()` - Get latest context
- `getMarketContextHistory(limit)` - Get history

### Performance
- `updateStrategyPerformance(performance)` - Update metrics
- `getStrategyPerformance(strategyId)` - Get performance
- `getTopStrategies(limit)` - Get top performers

### Analytics
- `getDashboardData()` - Get comprehensive dashboard
- `getSignalsReport(startDate, endDate)` - Get performance report
- `exportForTradingDivision(options)` - Export for trading bot

## 🐘 PostgreSQL Migration Notes

The schema is designed for easy migration to PostgreSQL:

1. Replace `INTEGER PRIMARY KEY AUTOINCREMENT` with `SERIAL` or `GENERATED ALWAYS AS IDENTITY`
2. Replace `JSON` with `JSONB` for better performance
3. Replace `BOOLEAN` with actual `BOOLEAN` type
4. Replace `DATETIME` with `TIMESTAMP WITH TIME ZONE`
5. Replace `DECIMAL` with `NUMERIC`
6. Consider TimescaleDB for time-series optimization
